## CAP (Certified Analytics Professional)

**Status:** Analyzed but NOT included. Here's why:

**The Good:**
- Legitimate credential from INFORMS (analytics professional body)
- Premium: ~$8,000-12,000 verified
- Respected in some industries (finance, healthcare)

**The Bad:**
- **Niche job market:** Only 80-120k jobs (8-15x smaller than cloud data)
- **Lower ROI:** $12,000 premium is 50% less than AWS Analytics ($32k)
- **Growing slowly:** +2-3%/year (vs. cloud data +36%/year)
- **Less relevant for military IT transition:** Requires statistics background, not common in IT roles
- **Overlaps with Google Analytics cert:** We chose GCP instead (higher demand, better ROI)

**Our Decision:** CAP didn't beat the cutoff. Better options existed with higher ROI and larger job markets.

---

## SQL Certification (Microsoft SQL Server, Oracle, etc.)

**Status:** Analyzed but NOT included. Here's why:

**The Reality:**
- SQL is a **skill, not a career certificate**
- It's a prerequisite (like knowing English as an engineer), not a specialization

**Why It Didn't Qualify:**
1. **No standalone credential** - No major certification body for "SQL skills"
   - Microsoft SQL Server: Part of Azure data engineer (we included Azure)
   - Oracle: Declining certification (Oracle cert market -3%/year)
   - PostgreSQL: No official certification

2. **Salary impact is unclear** - "Knowing SQL" doesn't add salary, but it's table stakes
   - Example: Data engineer job requires SQL. But SQL-only person isn't a data engineer.
   - Premium: +$0-3,000 if at all (no standalone data)
   - It's baked into the job requirement, not a premium credential

3. **Job market consideration:** Every data engineer knows SQL
   - It's like saying "knowing Excel" is a credential for accountants
   - You don't get paid extra for it; you're expected to have it

**Our Decision:** SQL is a skill within these certifications (all data certs require SQL), not a standalone credential. Including it would dilute the actual career-boosting certs.
