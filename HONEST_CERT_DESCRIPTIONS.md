# Honest Certification Descriptions (No Unsupported Claims)

## HIGHLY RELEVANT Tier
These are the core certifications for this role. Based on:
- Industry standard requirements for the role
- Common job posting requirements
- Skill alignment with role responsibilities

### Operations Management
- **Project Management Professional (PMP)**: Industry gold standard for project management. Widely recognized across industries for ops management roles.
- **Project+ (CompTIA)**: Entry-level project management certification. Good starting point for PMP or alternative if PMP not feasible.

### HR Management
- **Project Management Professional (PMP)**: Recognized in HR for managing organizational development, change management, and HR transformation projects.
- **Project+ (CompTIA)**: Entry-level alternative for HR project management and process improvement initiatives.

### Data Science / Data Analyst
- **GCP Data Engineer**: Industry-standard credential for data engineering and analytics roles using Google Cloud Platform.
- **AWS Analytics Specialty**: Demonstrates expertise in AWS analytics services, widely used in enterprise data roles.
- **Databricks Certified Engineer**: Validates hands-on expertise with Databricks lakehouse platform, increasingly important for modern data teams.

### Cyber/IT Operations
- **AWS Solutions Architect Associate**: Demonstrates cloud architecture knowledge for IT operations and infrastructure management.
- **Security+ (CompTIA)**: Foundational security certification, broadly applicable across IT operations and security roles.
- **Azure Administrator**: Cloud platform expertise for organizations using Microsoft Azure infrastructure.
